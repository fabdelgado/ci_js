Recursos para desarrolladores

  • Ruby - ci_uy: Gema en Ruby para validar C.I.s.
  • Python - ci_py: Módulo Python para validar C.I.s.
  • JavaScript - ci_js: Código JavaScript para validar C.I.s.
  • jQuery - ci - Plugin para validar cédulas en jQuery.
  • Node - ci_node - port en Node, instalable con NPM.
  • PHP - ci_php - port en PHP, instalable via composer.
  • Go - ciuy - port en Go.
  • Crystal - ci_uy - port de la versión en Ruby como shard en Crystal.
  • Prolog - ci_pl - Cláusulas para validar cédulas en Prolog.
  • Rust - ci-uy - port en Rust.
  • Java - ciuy - Librería en Java.
